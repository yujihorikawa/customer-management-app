# 顧客管理システム

## 概要

PHP・MySQLを使用して開発した顧客管理システムです。

営業職として培った顧客管理業務の経験をもとに、実際の業務で利用することを想定して制作しました。

顧客情報の登録・一覧表示・編集・削除に加え、検索機能や入力バリデーションを実装し、業務効率化を意識したシステムとなっています。

---

## 使用技術

* PHP
* MySQL
* PDO
* HTML
* CSS
* Git / GitHub

---

## 実装機能

* 顧客登録（Create）
* 顧客一覧表示（Read）
* 顧客編集（Update）
* 顧客削除（Delete）
* 顧客検索（Search）
* 入力バリデーション

---

## 工夫した点

### PDOを利用した安全なデータベース接続

PDOのプリペアドステートメントを利用し、SQLインジェクション対策を行いました。

### 検索機能の実装

顧客名や会社名から対象データを検索できる機能を実装し、業務での利用を想定した利便性を高めました。

### 入力バリデーション

登録・編集時に入力内容をチェックし、未入力や不正な電話番号を防止する仕組みを実装しました。

### 視認性を意識したUI

CSSを利用して見やすい一覧画面を作成し、操作しやすいレイアウトを意識しました。
