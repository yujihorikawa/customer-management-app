# 顧客管理システム

## 概要

PHP・MySQLを使用して開発した顧客管理システムです。

営業経験を活かし、実際の顧客管理業務を想定して制作しました。

## 使用技術

* PHP
* MySQL
* PDO
* HTML
* CSS

## 実装機能

* 顧客登録（Create）
* 顧客一覧表示（Read）
* 顧客編集（Update）
* 顧客削除（Delete）
* 顧客検索（Search）

## 工夫した点

* PDOを使用した安全なデータベース接続
* 検索機能による顧客情報の絞り込み
* CSSを使用した視認性の高いUI設計
