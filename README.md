# 顧客管理システム

## 概要

PHP と MySQL で作成した顧客管理システムです。

顧客情報の登録、一覧表示、編集、削除、検索に加えて、ログイン機能、メールアドレス管理、登録日時表示を実装しています。

営業職として培った顧客管理業務の経験をもとに、実際の業務で利用することを想定して制作しました。

## ログイン情報

デモ用の初期ログイン情報です。

- ユーザー名: `admin`
- パスワード: `password123`

ログイン情報は `auth_config.php` で変更できます。

---

## セットアップ

1. MySQL で `database.sql` を実行します。
2. `db.php` の接続情報を環境に合わせて変更します。
3. PHP が動作するローカル環境にファイルを配置します。
4. ブラウザで `login.php` にアクセスします。

---

## 既存テーブルを更新する場合

すでに `customers` テーブルがある場合は、`database.sql` 内の ALTER TABLE 文を参考に、以下のカラムを追加してください。

- `email`
- `created_at`

## 工夫した点

### PDO を利用した安全なデータベース接続

PDO のプリペアドステートメントを利用し、SQL インジェクション対策を行っています。

### 入力バリデーション

登録・編集時に、名前、電話番号、メールアドレス、会社名の入力内容をチェックしています。

### ログイン制御

未ログインの場合は顧客一覧や登録画面にアクセスできないようにし、セッションでログイン状態を管理しています。

### 検索機能

名前、電話番号、メールアドレス、会社名を対象に検索できます。

### 視認性を意識したUI

CSSを利用して見やすい一覧画面を作成し、操作しやすいレイアウトを意識しました。
